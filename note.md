``

``

- array shift

``
const user = ["Yusuf", "Shaikh", "Arman"];
user.shift();
console.log(user)
``

- array unshift - The unshift() method adds new elements to the beginning of an array.

``const user = ["Yusuf", "Shaikh", "Arman"];
user.unshift("yusuf");
console.log(user)
``

- map() - creates a new array from calling a function for every array element.
